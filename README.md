# e-plantShopping

**e-plantShopping** is an online shopping cart application for a virtual plant store called **Paradise Nursery**.  
The app allows users to browse different categories of house plants, add them to a shopping cart, and manage their purchases before checkout.

## Project Overview

This project is a frontend shopping cart application built to demonstrate dynamic UI updates, state management, and component-based design.  
Users can explore a variety of plants, view their details, add them to a cart, and adjust quantities in real time.

## Features

- **Landing Page**
  - Welcome screen with a button linking to the product listing page

- **Navigation Bar**
  - Links to:
    - Landing page  
    - Product listing  
    - Shopping cart  
  - Displays the number of items in the cart

- **Product Listing Page**
  - Displays plant cards with:
    - Image  
    - Name  
    - Description  
    - Price  
    - ‚ÄúAdd to Cart‚Äù button  
  - Plants are grouped into at least two sections, such as:
    - Aromatic Plants  
    - Medicinal Plants  

- **Shopping Cart Page**
  - Displays one card per plant type added to the cart
  - Each cart card includes:
    - Plant thumbnail  
    - Unit price  
    - Total price for that plant  
    - Buttons to increase and decrease quantity  
    - Delete button  

- **Cart Controls**
  - Continue Shopping button  
  - Checkout button  
  - Total cost updates dynamically when quantities change  

## Purpose

The **e-plantShopping** project applies the concepts learned in the practice project, including:
- Managing shared application state
- Updating UI based on user interactions
- Calculating totals dynamically
- Handling navigation between multiple pages

This project simulates a real online store experience while reinforcing modern frontend development skills.
